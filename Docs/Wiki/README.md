# Project Wiki Docs

This directory contains the active planning and requirements documentation migrated from local planning drafts.

Use these entry points:

- `Docs/Wiki/Home.md` for navigation
- `Docs/Wiki/requirements/Requirements-Index.md` for FR index
- `Docs/Wiki/nfr/NFR-Index.md` for NFR index
- `Docs/Wiki/Testing-Index.md` for test strategy and traceability
- `Docs/Wiki/Diagnostics-Index.md` for runtime diagnostics code registry
- `Docs/Wiki/Task-Runner.md` for command interface contract

Status fields are preserved from draft sources (`READY`, `DRAFT`, `Pending`, `Active`, `FINAL`) so teams can continue review and promotion in-repo.
